# :material-numeric-1-circle: mpv


### save time-point when playing a video using mpv 

```bash
mpv --save-position-on-quit YOUR-VIDEO.mp4
```

!!! tip
    can be added as an **alias** into your `~/.bashrc` file

```bash
alias impv='mpv --speed=1.1 --save-position-on-quit --osd-level=3 --osd-align-y=bottom --osd-align-x=right --osd-color=0/1/1  --osd-font-size=30'
```
